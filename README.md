# Moshify Clone

A fully responsive clone of the Moshify landing page built using HTML, CSS, and JavaScript. This project was created to practice modern frontend development, responsive layouts, animations, and clean UI design.

## Features
- 📱 Fully responsive design
- 🎨 Modern and clean user interface
- ✨ Smooth animations and transitions
- 🧭 Responsive navigation menu
- ⚡ Optimized HTML & CSS structure
- 💻 Built with vanilla JavaScript

## Technologies Used
- HTML5
- CSS3
- JavaScript (ES6)

## Purpose
This project was built for educational purposes to improve my frontend development skills by recreating a professional website design.

## Disclaimer
This is a personal learning project inspired by the Moshify website. All design credit belongs to the original creators.
